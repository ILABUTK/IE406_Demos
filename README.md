# IE406_Demos

This repository is a demo repo for the IE406 Simulation class at the University of Tennessee, Knoxville.

## Overview

IE406 is an Industrial Systems Engineering simulation course at UTK. This repo holds example AnyLogic models and demo scenarios used in class exercises, assignments, and student learning.

## Contents

- `Demo_MoveByTransporter/` - transporter movement example
- `Demo_My_Robot/` - robot cell demo
- `Demo_ProcessByRobot/` - process automation with robot demo
- `Demo_RollerCoaster/` - complex agent-based amusement ride model
- `Demo_Slotting Policies in Drive-in Storage/` - warehousing slotting policy demo
- `Demo_Tiny_MHL/` - small material handling line demo
- `Demo_Two_Level_Lift/` - two-level lift demo
- `Deno_Rotation in 3D - Sign on Two Posts/` - 3D rotation demonstration
- `Tutorial_GitHub/` - GitHub tutorial resources and setup

## Usage

1. Open an AnyLogic project file (`*.alp` or `*.alpx`) in AnyLogic.
2. Explore the model structure in the Projects pane.
3. Run experiments via the Experiments panel and inspect output charts/tables.

## Notes

- Models are class teaching examples and not production systems.
- Some models may not be compatible with all versions of AnyLogic; use a version close to what was used in class (e.g., AnyLogic 8.x) or test and adapt as needed.
- Adapt and extend for course projects and study.
